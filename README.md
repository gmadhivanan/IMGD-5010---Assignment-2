# Create your own Hero of Time
Link is the Hero of Time from the classis video game series the Legend of Zelda.
The first Legend of Zelda debuted in the US in the late 80s on the Nintendo Entertainment System and the latest game in the main series The Legend of Zelda the tears of the Kingdom. All of the main series game center around the main character Link gaining the power of the Triforce to defeat evil and save the kingdom along with the princess Zelda. <br>
